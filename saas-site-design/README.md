# saas-site-design

Produces a comprehensive design specification — layout patterns, typography, color system, component specs, interaction design, and responsive strategy — for implementing a SaaS marketing site. Takes marketing site content and turns it into a buildable visual system reviewed through three lenses: strategy, conversion, and buildability.

## Install

```bash
npx skills add https://github.com/yarlson/skills/tree/main/saas-site-design
```
