# prd-vertical-slicer

Convert PRDs (including phase-based PRDs) into adaptive vertical slice plans with architecture guardrails and tech-debt prevention. Produces sequenced, user-visible slices — not tasks, not component phases — with explicit debt ledgers and stabilization outcomes.

## Install

```bash
npx skills add https://github.com/yarlson/skills/tree/main/prd-vertical-slicer
```
